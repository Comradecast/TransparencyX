# Project Brief

## Goal
Build a Python civic-data project that consolidates U.S. congressional financial disclosure information into a normalized, searchable dataset.

## Initial Scope
- House and Senate members/candidates financial disclosures
- Business ownerships
- Outside income
- Assets
- Liabilities
- Positions held
- Estimated low/high value bands from disclosure ranges

## Principles
- No aggressive scraping.
- No unofficial sources in initial phases.
- No over-engineering. Keep dependencies and architecture simple.
- Strict adherence to official range labels without projecting precision.
